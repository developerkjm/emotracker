Tracker 기록 웹앱

매일을 기록하고, 캘린더에서 흐름을 볼 수 있는 나만의 다이어리

🛠️ 사용 기술 스택

  • 백엔드: Spring Boot, Spring Security, JPA (Hibernate), MySQL
  • 프론트엔드: Thymeleaf, HTML5, CSS3, JavaScript, FullCalendar.js
  • 인증: 일반 회원가입/로그인 ( 추가예정 + Google OAuth2 로그인 )
  • 파일업로드: Multipart + 로컬 저장, AWS S3 연동 예정
  • 배포: AWS EC2, RDS, S3 (진행중)
  • 기타: GitHub, Maven, Lombok

✨ 주요 기능

  • 일별 감정 등록, 수정, 삭제 (회원전용)
  • 커뮤니티 게시판: 게시글/댓글/대댓글 기능, 파일 첨부 (회원전용)
  • 캘린더에서 감정 흐름 시각화
  • 작성자 IP 앞자리 표시 (관리자만 전체 확인 가능)
  • 회원가입 / 로그인 / 로그아웃
